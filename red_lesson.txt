Гитхаб — в моем понимании это сайт для программистов на котором они могут делится своими кодами, идеями, попросить найти ошибки которые они сами не могут выявить. Так же он может использоваться для хранения своих кодов и показать их обществу.

В силу моих малых знаний	я не знаю какую-либо альтернативу гитхабу.

Гит — помогает отслеживать историю изменений в файлах.

git init — используется для создание нового и пустого репозитория Git в текущей директории

git commit — сохраняет текущие изменения в репозитории 

git branch — управление ветками(? Не знаю что это) в репозитории 

git merge — объединение изменений из одной ветки в другую 

git push — отправка локальных коммитов на  удаленный репозиторий 

git remote add — добавление удаленного репозитория
